Terms of Service Privacy Policy

Last updated: June 16, 2026

🔒 Short version: Melodix collects minimal data necessary to function. We do not sell or share your data with third parties.

1. Information We Collect

Melodix collects only the data strictly necessary to provide its features:

  • Guild (Server) ID — to store per-server settings such as DJ role and configuration
  • User ID — temporarily, to attribute commands, manage votes, and deliver reminders during an active session
  • Text channel ID — to know where to send bot responses and notifications
  • Message content — read only to process bot commands; never stored or logged
  • Command inputs — such as search queries or poll options, processed in real time and not stored persistently
  • Giveaway data — host User ID, prize description, list of participant User IDs, and end time, stored persistently until the giveaway concludes or is manually deleted
  • Welcome settings — welcome channel ID, auto-assign role ID, custom welcome message, and welcome image URL, stored persistently per server until changed or the bot is removed

2. Information We Do NOT Collect

  • Personal information such as email addresses, real names, or phone numbers
  • Voice data or audio recordings of any kind
  • Message content outside of direct bot commands
  • Payment or financial information
  • Browsing history or data from outside Discord

3. How We Use Your Data

The data collected is used solely to:

  • Process and fulfil commands (music, moderation, polls, reminders, etc.)
  • Save server-specific settings persistently (e.g. DJ role)
  • Deliver reminders to users who requested them
  • Manage session state during active bot use

4. Data Storage

The following data is stored persistently in JSON files on the bot's host server:

  • Server configuration settings (DJ role, prefix, request channel, etc.)
  • Welcome settings (channel, role, message, image)
  • Active giveaway data (host, prize, participants, end time) — deleted automatically when the giveaway ends or is cancelled

Session data such as music queues, active votes, and reminders exist only in memory and are lost when the bot restarts. No external database or analytics service is used.

5. Third-Party Services

Melodix interacts with the following third-party services:

  • YouTube — search queries may be sent via Lavalink for music playback
  • SoundCloud — used as a primary audio source
  • Spotify — used only to retrieve track metadata (title, artist); no audio is streamed from Spotify directly
  • Genius — used to fetch song lyrics on request

Your use of these platforms is governed by their own privacy policies.

6. Data Retention

Server configuration data is retained until manually changed or the bot is removed from the server. All session data is discarded when the bot restarts or disconnects.

7. Your Rights

You may request deletion of any stored data related to your server at any time by contacting us via the Melodix support server. Removing the bot from your server will also cause all associated settings to become inaccessible.

8. Children's Privacy

Melodix does not knowingly collect data from users under the age of 13. If you believe a minor has used the bot, please contact us so we can address the situation appropriately.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Continued use of the bot after changes constitutes acceptance of the updated policy.

10. Contact

For any privacy-related questions, please reach out via the Melodix support server on Discord.
